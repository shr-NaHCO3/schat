# id
字符串，`8-32`个字符。仅包含字母、数字。
用于区分用户及登录。

&nbsp;

# nickname
普通的字符串，`8-32`个字符。禁止奇怪的字符，包括但不限于 `null undefined \0`。
用于让用户拥有他们喜爱的名字。

&nbsp;

# password

用于登录等
